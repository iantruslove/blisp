# blisp

browser lisp

A browser-based Scheme-ish repl, written in JavaScript.

## Getting Started
Look at this code running live at http://iantruslove.github.com/blisp/

## Documentation

### Supported Expressions

#### Booleans

```
#t
=> true

#f
=> false
```

#### Basic Arithmetic

```
(+ (* 2 3) (- 10 (/ 4 2)))

=> 14
```
